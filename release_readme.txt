Kazesawa Font
===============

Kazesawa Font in GitHub: https://github.com/kazesawa/kazesawa

Kazesawa Font は、M+ と Source Sans Pro の合成フォントです。

ひらがな、カタカナと一部の記号類に Migu 1C と同じ可変幅化処理を行なっています。
